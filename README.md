## Requirements

Run the npm install to add the dependencies

```shellscript
npm install
```

## Run

Run the main code functionality:

```shellscript
tsc && node dist/index.js
```

## Test

Run the test for all logic methods:

```shellscript
npm run test
```

## Build

First, build the app:

```sh
npm run build
```

Then run the app production mode:

```sh
node dist/index.js
```
